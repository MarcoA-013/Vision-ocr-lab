# Vision-ocr-lab

## Descrição
Testando o Azure Vision Studio, aqui na modalidade OCR, para extrar texto de imagens.

## Pré-requisitos
- Python 3.6 ou superior
- Conta no Azure
- Chave de acesso ao Azure Vision Studio
- Google Colab
